Usage Instructions
  1. Enter the path to a SWASH directory containing TOXSWA projects.
  2. The directory will be scanned for project folders containing 'toxswa' subfolders.
  3. Multi-select one or more project folders and click 'Extract Data'.
  4. Each project should contain a 'toxswa' folder with .sum files.
  5. Extracted data includes PEC values for water and sediment compartments.
  6. Use the RAC value to highlight exceedances in the table.
  7. Choose between Parent and Metabolite compound types.
  8. Sort data by filename, compound, or scenario.
  9. Enable areic comparison to highlight mitigation scenarios.
  10. Click the copy button to copy selected data or the entire table.
  11. Click 'Export to Excel' to save the data with detailed sheets.
  12. Click the reset button to clear the table.
